# 🌐 Projekty Sieciowe / Network Projects

Kolekcja zaawansowanych projektów związanych z inżynierią sieci, konfiguracją urządzeń Cisco oraz wdrażaniem rozwiązań bezpieczeństwa. Wszystkie projekty zostały stworzone w **Cisco Packet Tracer** i demonstrują umiejętności na poziomie **CCNA**.

---

## 📂 Lista projektów:

### 1. [Enterprise Multi-Site Network](./Enterprise-Multi-Site-Network/)
Kompleksowa sieć korporacyjna z segmentacją VLAN, dynamicznym routingiem OSPF, firewallem ASA oraz infrastrukturą serwerową (DHCP/DNS/HTTP).

**Kluczowe technologie:**
- Inter-VLAN Routing (Switch L3)
- OSPF Multi-Area Routing
- ASA Firewall z ACL
- DHCP Relay (IP Helper)
- Hierarchiczny design (3-tier architecture)

**Poziom trudności:** ⭐⭐⭐⭐ (Zaawansowany)
